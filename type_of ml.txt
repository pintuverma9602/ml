Definition: The model learns from labeled data (input and corresponding output).

Algorithms:

Linear Regression

Logistic Regression

Decision Trees

Classification: Predict categories (e.g., spam/ham, disease/no disease)

Anomaly Detection

Algorithms:

K-Means Clustering

Hierarchical Clustering

Definition: Uses a small amount of labeled data and a large amount of unlabeled data.

Goal: Improve learning accuracy when labeled data is scarce.

Examples:

Medical Diagnosis (few labeled patient reports)

Web Page Classification

Techniques:

Self-training

Co-training

Graph-based methods

Definition: The model learns through trial and error to maximize a reward.

Goal: Learn a sequence of decisions by interacting with an environment.

Key Concepts:

Agent: Learner/decision-maker

Environment: What the agent interacts with

Reward: Feedback signal for actions

Examples:

Game Playing (e.g., AlphaGo, Chess)

Robotics

Autonomous Vehicles

Algorithms:

Q-Learning

Deep Q-Networks (DQN)

Policy Gradient Methods
